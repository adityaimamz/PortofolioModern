# Portfolio

This project is an interactive and responsive portfolio built with Next.js and React, featuring advanced 3D components and smooth animations.

## 🚀 Tech Stack

This project utilizes the best modern libraries to ensure high performance and a premium visual experience:
- **Core Framework**: [Next.js 14](https://nextjs.org/) & [React 18](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation & UI**: [Framer Motion](https://www.framer.com/motion/), [Lottie](https://lottiefiles.com/), [Lucide React](https://lucide.dev/), [React Icons](https://react-icons.github.io/react-icons/)
- **3D Rendering**: [Three.js](https://threejs.org/), [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction), [@react-three/drei](https://github.com/pmndrs/drei), and [Three-Globe](https://github.com/vasturiano/three-globe)
- **Deployment**: [Vercel](https://vercel.com/)
- **Utilities**: Sentry (Monitoring), clsx & tailwind-merge (CSS class management).

## ✨ Key Features

- **Modern Design:** Features a premium aesthetic with dark mode and exceptional UI integration.
- **Smooth 3D & Animations:** Powered by Framer Motion and Three.js, creating a highly immersive user experience.
- **Fully Responsive:** Looks stunning on all devices, from mobile phones to a desktop screen.

## 🛠️ Getting Started

Follow these steps to run the project locally:

1. **Install Dependencies:**
   Open a terminal in the project folder and run:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

2. **Run the Development Server:**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

3. **View the Result:**  
   Open your browser and visit [http://localhost:3000](http://localhost:3000) to view your portfolio site. You can start editing the page directly with React, and it will auto-update instantly thanks to Fast Refresh.

## 📜 Available Scripts

In the `package.json` file, several standard scripts are provided:

- `npm run dev`: Runs the app in the development mode.
- `npm run build`: Builds the app for production usage.
- `npm run start`: Starts the production server based on the output build.
- `npm run lint`: Runs the linter to catch syntax errors or inconsistencies.
- `npm run deploy`: Quick command to deploy the application (set to Vercel CLI: `vercel --prod`).
